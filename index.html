This app is installed as a fullscreen PWA via GitHub Pages. If Chrome shows an install banner — tap Install. To reinstall: tap ⋮ → Add to Home Screen in Chrome.

All data is stored locally on this device. No internet connection is needed for attendance — only for the first load of the Excel export library.
