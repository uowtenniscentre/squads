📊 Export for Statistics Viewer
Export a JSON file for the stats.html statistics app. Open stats.html in any browser, drag-drop this file in, and get full charts — season trend, squad comparison, group overview and player heatmaps.
↗ Open Stats Viewer
⬇️ Archive Export (Excel)
Full attendance archive — one tab per programme group plus Summary and Coach Notes. Use for record-keeping and sharing with club management. Needs internet once to load the Excel library.
📋 Import Squads from Excel
Upload an Excel file with a sheet named Squads with columns: Group, Squad name, Coaches, Weekday. Only new squads are added — existing ones are not overwritten.
👦 Import Players from Excel
Upload the same Excel file with a sheet named Players with columns: ID, Last name, First Name, Birth Year, Gender, Squad1…Squad5. Only new players are added — existing ones (matched by ID or name) are not overwritten.
